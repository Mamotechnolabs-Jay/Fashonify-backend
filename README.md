### Installation ###

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2. Navigate to the project directory:
    ```sh
    cd fashionify-backend
    ```
3. Install dependencies:
    ```sh
    npm install
    ```
4. Start the application:
    ```sh
    npm start
    ```
